Gerencidador de tarefas

Este projeto foi desenvolvido para facilitar o gerenciamento de tarefas pessoais ou de equipe. Utilizando PHP para a lógica de back-end e MySQL para o armazenamento de dados, o sistema permite aos usuários criar, editar, excluir e visualizar tarefas em tempo real. A interação dinâmica é garantida pelo uso de AJAX, proporcionando uma experiência fluida sem a necessidade de recarregar a página. A interface foi criada com HTML e CSS, garantindo uma navegação intuitiva e responsiva. Este projeto demonstra habilidades em desenvolvimento full stack, com foco em interações dinâmicas e banco de dados relacional.

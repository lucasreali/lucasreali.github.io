Projetos

Gerenciador de tarefas link

Este projeto foi desenvolvido para facilitar o gerenciamento de tarefas pessoais ou de equipe. Utilizando PHP para a lógica de back-end e MySQL para o armazenamento de dados, o sistema permite aos usuários criar, editar, excluir e visualizar tarefas em tempo real. A interação dinâmica é garantida pelo uso de AJAX, proporcionando uma experiência fluida sem a necessidade de recarregar a página. A interface foi criada com HTML e CSS, garantindo uma navegação intuitiva e responsiva.

Calculador Web - link

Uma aplicação web de calculadora desenvolvida com JavaScript, CSS e HTML. Esta calculadora realiza operações básicas, como adição, subtração, multiplicação e divisão, e inclui um histórico do último valor digitado, permitindo ao usuário revisar facilmente suas entradas recentes. Este projeto destaca habilidades de manipulação de DOM com JavaScript e criação de interfaces responsivas com CSS e HTML.
